# 📱 Spring Boot & REST APIs: Portfólio de Integração e Arquivos

Este repositório foi desenvolvido com o objetivo de consolidar conceitos fundamentais de desenvolvimento web com Spring Boot e sua respectiva integração com arquivos locais de dados. Ele funciona como um portfólio acadêmico, reunindo diferentes cenários de negócios, manipulação de fluxos I/O e serialização JSON.
