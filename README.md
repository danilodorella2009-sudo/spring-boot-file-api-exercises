# 📱 Spring Boot & REST APIs: Portfólio de Integração e Arquivos

Este repositório foi desenvolvido com o objetivo de consolidar conceitos fundamentais de desenvolvimento web com Spring Boot e sua respectiva integração com arquivos locais de dados. Ele funciona como um portfólio acadêmico, reunindo diferentes cenários de negócios, manipulação de fluxos I/O e serialização JSON em um único projeto unificado.

---

## 📌 Índice de Desafios no Projeto

### 1. API da Playlist 🎵
* **Conceitos Focados:** `@RestController`, `@GetMapping`, `@PostMapping`, `FileReader`, `BufferedReader`.

### 2. O Catálogo de Personagens Web 🎮
* **Conceitos Focados:** `Herança`, `Polimorfismo`, `@PathVariable`, `CSV Split`, `instanceof`.

### 3. O Portal do Hackathon 🏆
* **Conceitos Focados:** `Validação de Regras`, `BufferedWriter`, `Geração de Logs/Relatórios`, `JSON Customizado`.
